Cómo abrirlo

Como web pública o en local.

En GitHub Pages

Sube este archivo como index.html, conserva la carpeta data/ y activa Pages. La app leerá documents.json desde el propio repositorio.

En tu ordenador

Ejecuta python3 -m http.server en la carpeta del proyecto y abre http://localhost:8000. No uses doble clic si quieres que el navegador cargue bien el JSON local.

Versión pensada para GitHub Actions: scraping en la nube, visualización en la web.
